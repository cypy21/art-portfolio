# Art Portfolio Website

A modern, professional art portfolio website built with Next.js 14, React, and Tailwind CSS with a beautiful dark blue theme.

## Features

- 🎨 Dark Blue Theme with bright sky blue accents
- 📱 Fully Responsive design
- ✨ Smooth Animations with Framer Motion
- 🖼️ Gallery Section with filtering
- 📧 Contact Form
- ⚡ Optimized Performance

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Customization

Edit these files:
- **Your info:** `components/About.tsx`
- **Gallery:** `components/Gallery.tsx`
- **Colors:** `tailwind.config.js`
- **Contact:** `components/Contact.tsx`
